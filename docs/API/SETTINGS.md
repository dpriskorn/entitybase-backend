# Settings API

> API endpoint for application settings

---

## GET /settings
**Get Settings**

Get current application settings (excludes sensitive values).

| Aspect | Details |
|--------|---------|
| Operation ID | `get_settings_settings_get` |
| Method | `GET` |
| Path | `/settings` |
| Response | Description |
|----------|-------------|
| 200 | Successful Response |
